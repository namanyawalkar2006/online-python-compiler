# 🚀 Zydex-PDF 
**Next-Gen Mobile-Inspired PDF Viewer for Desktop**

Zydex-PDF is a lightweight, high-clarity PDF reader built with Python. It brings the smooth, floating-UI experience of a mobile app to your desktop.

## ✨ Key Features
* 🌙 **Night Vision:** High-contrast inverted rendering for late-night reading.
* 🔍 **Smart Zoom:** Contextual zooming that stays sharp at 400%.
* 🖐️ **Kinetic Panning:** Drag-and-swipe document navigation.
* 🎨 **Modern UI:** Built with CustomTkinter for a sleek, OLED-black aesthetic.

## 🛠️ Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Zydex-PDF.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python main.py`
